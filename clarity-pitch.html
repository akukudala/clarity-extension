The Solution

Clarity makes manipulation
visible — and a little
uncomfortable.

01 — DETECT
Scrapes the real text
Our content script walks every text node on the page, identifies urgency, scarcity, social proof, and promotional pressure language using a deep regex library.
02 — TRANSLATE
Sends it to Claude
The actual on-page text is batched and sent to Claude Haiku — which rewrites each one as what it actually means in a warm, sardonic, one-sentence callout.
03 — SURFACE
The pause effect
A Clarity card slides in from the bottom-right. The original text is crossed out. The honest rewrite sits beneath it. You pause. You rethink. You decide.
04 — RESPECT
Non-intrusive by design
Cards auto-dismiss after 12 seconds. Nothing is blocked or replaced on the page. Clarity informs — it doesn't become the thing that hijacks your attention next.

Technical Architecture

Built in one night.
Ships on real sites.

Frontend Layer
⚙️
Chrome Extension MV3
Content scripts for DOM scraping. Service worker for API calls and in-memory caching. MutationObserver for SPA navigation.
🔍
TreeWalker + Regex Scraper
Walks all text nodes after 5s delay (for dynamic content). Matches 20+ manipulation patterns across urgency, scarcity, social proof, and promo categories.
🎨
Injected Overlay Cards
Zero-dependency UI. Staggered 1.5s apart. Original text crossed out, rewrite below. Auto-dismiss at 12s with slide-out animation.
Intelligence Layer
🤖
Claude Haiku
Fast, cheap, and surprisingly witty. System-prompted to write like "a knowing friend whispering the truth" — never preachy, always on point.
⚡
Batched Prompting
All scraped texts sent in one API call. Claude returns a numbered list. One round-trip per page visit.
🛡️
In-Memory Cache
Service worker caches responses by domain + text fingerprint. Repeat visits get instant rewrites with no extra API cost.
